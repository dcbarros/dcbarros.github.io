Sobre mim

Olá, meu nome é Davison, e estou aqui para compartilhar um pouco sobre minha jornada no desenvolvimento de software. Tudo começou em 2016, quando participei do projeto Inova Talentos com Python e criei uma aplicação complexa para uma empresa pública em meu estado. Desde então, tenho me dedicado ao desenvolvimento web e estou sempre buscando aprimorar minhas habilidades.

Durante meu curso de Análise e Desenvolvimento de Sistemas na PUC Minas, aprofundei minha paixão pelo desenvolvimento de software e contribuí para diversos projetos, como AppPoint e PetNow. Atualmente, estou focado em expandir meus conhecimentos em Java por meio do curso Santander Coders, onde estou me aprofundando em Java e sobre suas principais frameworks como o SpringBoot.

Além do desenvolvimento de software, tenho outros interesses, como jogar videogames e jogos de tabuleiro. Acredito que essas atividades me ajudam a manter a mente afiada e criativa, o que é essencial no mundo da tecnologia.

Projeto API Agenda Médica

Desenvolvido como parte do módulo de Programação Web II do Santander Coders, este projeto de agendamento de consultas médicas foi realizado em equipe. Utilizamos tecnologias como Spring Boot, Docker, PostgreSQL e Redis, sob orientação dos professores. A API oferece funcionalidades distintas para pacientes, médicos e administradores, com permissões adequadas a cada tipo de usuário. Durante o desenvolvimento, exploramos conceitos fundamentais de arquitetura REST, aprofundamos nosso conhecimento em Spring Boot e implementamos segurança com Spring Security e tokens JWT. Este projeto não apenas resultou em uma aplicação funcional, mas também proporcionou entendimento sobre aspectos cruciais de segurança e arquitetura em sistemas RESTful.

Projeto de Extensão - Hospital Eduardo de Menezes

Um projeto de desenvolvimento de software para avaliação fisioterapêutica em ambiente hospitalar busca solucionar problemas enfrentados pelos fisioterapeutas da instituição parceira. Atualmente, eles preenchem formulários em papel, o que pode levar a erros, falta de segurança nos dados e dificuldade de compartilhamento. O objetivo principal é criar uma plataforma mobile que facilite o preenchimento, automatize o cálculo do score, e armazene os registros online com segurança, proporcionando maior praticidade e qualidade no atendimento. Essa iniciativa visa melhorar a eficiência operacional, garantir a precisão nos cálculos, facilitar o acesso aos dados e reforçar a segurança das informações, resultando em uma transformação significativa na qualidade dos serviços de fisioterapia e benefícios diretos para pacientes e profissionais de saúde envolvidos.

Projeto de consultas médicas - Apppoint

Com o objetivo de intermediar o agendamento de consultas médicas entre pacientes e clínicas, este projeto propõe a criação de uma plataforma para a web em que seja possível encontrar médicos, verificar suas agendas e, finalmente, realizar um agendamento. Da parte das clínicas, esta mesma plataforma proporcionará o acesso a um dashboard de gerenciamento de seus médicos e respectivas agendas, bem como a possibilidade de integrar pacientes tradicionais (aqueles que não utilizam a plataforma) à agenda virtual.

MedVaz - Projeto INOVA Talentos

As empresas de distribuição de gás natural adotam uma rede de gasodutos subterrâneos, usando para isso métodos de perfuração não-destrutivos, diminuindo os transtornos ocasionados pelo método destrutivo. Entretanto, a metodologia adotada possui um lado desfavorável, uma vez que os gasodutos são construídos em Polietileno de Alta Densidade (PEAD) - menos resistentes a impactos mecânicos – há grande risco de acidentes envolvido. Uma vez que esses dividirem espaço com instalações de outras utilidades, a intervenção e manutenção de tais instalações podem ocasionar numa eventual perfuração dos gasodutos. Neste cenário, foi elaborado um software para contabilidade das perdas de gás com o auxílio de processamento de imagem o qual verificou-se uma boa precisão na quantificação dessas, sendo observado experimentalmente essa eficácia.
